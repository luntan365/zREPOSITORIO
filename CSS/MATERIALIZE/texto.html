Um defeito comum que vemos em vários frameworks é a falta de suporte para textos realmente responsivos. Enquanto os elementos na tela respondem fluidamente, o texto ainda se redimensiona em uma base fixa. Para melhorar este problema, em páginas carregadas de texto, nós criamos uma classe que redimensiona fluidamente o tamanho do texto e a altura da linha para otimizar a leitura do usuário. O tamanho da linha está entre 45-80 caracteres e a altura da linha se redimensiona para ser maior em telas menores. Para ver o Fluxo de Texto em ação, redimensione seu navegador lentamente e observe o tamanho do texto mudar. Use o botão acima para trocar entre ligado/desligado o fluxo de texto e veja a diferença!

Para formatar linhas longas de texto em formato de elipse, adicione a classe truncate no campo que contém o texto. Olhe o exemplo abaixo onde o cabeçalho é formatado dentro de um cartão.
